# Stage 1: Your First AI Coding Session

Welcome! You're about to experience coding with AI assistance for the first time (or see it in a new way).

## What You'll Do (22 minutes)

- **Demo** (8 min): Watch the instructor code with AI
- **Your Turn** (11 min): Add a feature using AI
- **Reflect** (3 min): What just happened?

## The Goal: Experience AI Coding

Forget everything complicated. Here you'll just:
1. Open a file
2. Ask AI to add something
3. See it work

That's it. No theory, no best practices yet. Just experience the **vibe** of coding with AI.

## Demo: Watch First

The instructor will show you how to:
- Open an AL file
- Ask AI to add a simple feature (like a discount field)
- Accept the suggestions
- See it compile

**Key observation**: AI writes code fast. Really fast.

Keep it simple:

```

```

Then just accept what AI suggests!
