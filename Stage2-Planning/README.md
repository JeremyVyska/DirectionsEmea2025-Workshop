## What to Expect

AI is helpful but has limitations. You might see it catch:
- Style issues (formatting, naming)
- Obvious problems (hardcoded values, missing validation)
- General best practices

But it might miss:
- Platform-specific patterns
- Performance implications
- Deep architectural concerns

## Why Does This Matter?

Understanding what AI can and can't do helps you:
- Know when to trust AI suggestions
- Know when to dig deeper yourself
- Set realistic expectations for AI tools
